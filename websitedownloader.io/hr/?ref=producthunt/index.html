Web Downloader Značajke

Web Downloader, Website Copier ili Website Ripper omogućuje vam da preuzimanje web stranice s Interneta na lokalni tvrdi disk na svoje računalo. Web Downloader dogovara preuzetu stranice od strane izvornih web stranice u odnosu link-strukture. Preuzeta web stranici mogu se pronaći tako da otvorite jednu od HTML stranice u pregledniku.

HTML

The web grabilo uzima svaku HTML datoteku i download i klonovi ga na vašem lokalnom tvrdom disku. Možete koristiti HTML editor napraviti izmjene na svakoj html datoteke lokalno ili koristiti HTML editor online. U slučaju da niste koder možete koristiti WYSIWYG-HTML uređivač umjesto.

Link konverzija

Mi pretvoriti sve veze u HTML datoteke, tako da oni rade na lokalnoj razini, off-line, umjesto da pokazuje na online web stranice.

CSS i JavaScript

CSS i JavaScript datoteke preuzet će se pod jednostavnim strukture mape i pravilno navodi u HTML datoteke.

Slike

Kao i ostale imovine, Također će slike preuzeti i upućuje na lokalnoj razini.
