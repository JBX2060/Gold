Website Downloader Mga Tampok

website Downloader, website Copier o website Ripper pinapayagan ka na mag-download mga website mula sa Internet sa iyong lokal na hard drive sa iyong sariling computer. Website Downloader nag-aayos na-download na site sa pamamagitan ng ang orihinal na mga website na kamag-anak link-istraktura. Na-download na website ay maaaring browse sa pamamagitan ng pagbubukas isa sa mga pahina ng HTML sa isang browser.

HTML

ang web grabber tumatagal sa bawat file na HTML at mga pag-download at panggagaya ito sa iyong lokal na hard drive. Maaari kang gumamit ng HTML editor gumawa ng mga pagbabago sa bawat html file nang lokal o gumamit ng isang HTML editor online. Sa kasong ikaw ay hindi isang coder maaari kang gumamit ng WYSIWYG-HTML Editor sa halip.

link Conversion

convert namin lahat ng mga link sa mga file ng HTML upang ang gumagana ang mga iyon nang lokal, off-line, sa halip na tumuturo sa isang online na website.

CSS & JavaScript
