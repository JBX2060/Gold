Website-Downloader Funktionen

Website-Downloader, Website Copier oder Website Ripper ermöglicht es Ihnen, Webseiten aus dem Internet auf die lokale Festplatte auf Ihrem eigenen Computer herunterladen. Website-Downloader ordnet die heruntergeladene Website durch die ursprünglichen Webseiten relativen Link-Struktur. Die heruntergeladene Webseite kann durch Öffnen eines der HTML-Seiten in einem Browser durchsucht werden.

HTML

Das Web-Grabber nimmt jede HTML-Datei heruntergeladen und Klone es auf Ihrer lokalen Festplatte. Sie können ein verwenden HTML-Editor um Änderungen an den einzelnen HTML-Datei lokal oder verwenden Sie ein HTML-Editor Online. Falls Sie kein Programmierer Sie verwenden können WYSIWYG-HTML-Editor stattdessen.

Link Conversion

Wir wandeln alle Links in den HTML-Dateien, so dass sie vor Ort arbeiten, offline, statt auf einer Online-Webseite zeigen.

CSS & JavaScript

CSS und JavaScript-Dateien werden unter einem einfachen Ordnerstruktur und referenziert korrekt in den HTML-Dateien heruntergeladen werden.

Bilder

Wie andere Vermögenswerte, auch werden die Bilder heruntergeladen und lokal verwiesen.
