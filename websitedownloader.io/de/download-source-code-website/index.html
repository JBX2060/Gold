Wie Quellcode einer Website zum Download?

Updated On: September 20, 2016
Written By: Debjit Saha

Um eine Website alles, was Sie herunterladen müssen, ist die Website-Adresse (URL, für zB. apple.com oder google.com), eine Internetverbindung und WebsiteDownloader.io

WebsiteDownloader.io ist ein großes Werkzeug, mit dem Sie Laden Sie den Quellcode einer Website die auch die HTML-Dateien, statische Vermögenswerte wie JS (Javascript), CSS, Bilder und PDF-Dokumente.

Alles was Sie tun müssen, ist die Website-URL, in den Sie herunterladen möchten eingeben WebsiteDownloader.io und nach ein paar Minuten, in Abhängigkeit von der Größe der Website, Sie werden eine zip erhalten, die den HTML-Quellcode der Website enthalten würde.

Sie können auch die Liste der Dateien in der Vorschau, die das zip so würde beinhalten, dass Sie in vollem Umfang von den Inhalt der ZIP-versichert sind.

Das heisst, es gibt 1 Website für jeden 7 Menschen auf der Erde. Obwohl, es funktioniert nicht genau so, aber es würde so nicht falsch sein, um eine Schätzung machen – 1 Website für jeden 7 Menschen!
