Hauptvorteile

Der einfachste Weg zu Download von archive.org, die auch als Wayback Machine bekannt

Plattformunabhängig

Die Web-basierte Schnittstelle ermöglicht es Ihnen, zu verwenden, Wayback Machine Downloader gerade in Ihrem Browser auf jedem Betriebssystem und ohne Download oder jede Software zu konfigurieren.

Schnelle Vorschau

Wayback Maschine Downloader bietet eine schnelle Vorschau des Download-Ergebnis auf unseren Servern gehostet, ohne den Computer wertvollen Plattenplatz belegt.

Einfachheit

Wayback Maschine Downloader ist super einfach und leicht zu bedienen, doch es kommt mit erweiterten Funktionen wie nur ein Unterverzeichnis oder bestimmte Seiten von einer Website herunter (als ein Web-Seite Downloader).

Wayback Machine Downloader Eigenschaften

Wayback Machine Downloader ermöglicht Ihnen, Webseiten von archive.org auf der lokalen Festplatte auf Ihrem eigenen Computer herunterladen. Wayback Downloader ordnet die heruntergeladenen Website von den ursprünglichen Webseiten relativen Link-Struktur. Die heruntergeladene Webseite kann durch Öffnen eines der HTML-Seiten in einem Browser durchsucht werden.

HTML

Wayback Downloader nimmt jede HTML-Datei heruntergeladen und Klone es auf Ihrer lokalen Festplatte.

Link Conversion

Wir wandeln alle Links in den HTML-Dateien, so dass sie vor Ort arbeiten, offline, statt auf einer Online-Webseite zeigen.

CSS & JavaScript

CSS und JavaScript-Dateien werden unter einem einfachen Ordnerstruktur und referenziert korrekt in den HTML-Dateien heruntergeladen werden.

Bilder

Wie andere Vermögenswerte, auch werden die Bilder heruntergeladen und lokal verwiesen.
