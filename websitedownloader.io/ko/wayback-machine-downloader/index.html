뒤로 머신 다운: 어떻게 다운로드 웹 사이트 30 을 통해 두 번째 워크

주요 이점

에 가장 쉬운 방법 archive.org에서 다운로드, 또한 뒤로 시스템으로 알려져있다

플랫폼 독립적

웹 기반 인터페이스를 사용 할 수 있습니다 뒤로 기계 다운 바로 모든 운영 체제에 브라우저 및 다운로드하거나 소프트웨어를 구성하지 않고.

빠른 미리보기

뒤로 머신 다운 로더는 컴퓨터의 귀중한 디스크 공간을 사용하지 않고 우리의 서버에서 호스팅 다운로드 결과의 빠른 미리보기를 제공합니다.

간단

뒤로 머신 다운 슈퍼 간단하고 사용하기 쉬운, 아직는 단지 웹 사이트에서 하위 디렉토리 또는 특정 페이지를 다운로드 같은 고급 기능을 제공 (로 웹 페이지 다운).

뒤로 머신 다운 특징

뒤로 머신 다운 로더는 자신의 컴퓨터에 로컬 하드 드라이브에 archive.org에서 웹 사이트를 다운로드하실 수 있습니다. 뒤로 다운은 원래의 웹 사이트 상대적인 링크 구조에 의해 다운로드 된 사이트를 배치. 다운로드 된 웹 사이트는 브라우저에서 HTML 페이지 중 하나를 개방함으로써 탐색 될 수있다.

HTML

뒤로 다운 로컬 하드 드라이브에 각 HTML 파일과 다운로드 및 복제를한다.

링크 변환

그들은 로컬로 작동 할 수 있도록 우리는 HTML 파일에있는 모든 링크를 변환, 오프라인, 대신 온라인 웹 사이트를 가리키는.

CSS 및 자바 스크립트

CSS 파일 및 자바 스크립트는 간단한 폴더 구조에 따라 다운로드하여 HTML 파일을 올바르게 참조 될.

이미지

다른 자산과 마찬가지로, 또한 이미지를 다운로드하고 로컬 참조.
