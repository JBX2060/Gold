웹 사이트 다운 로더의 특징

웹 사이트 다운, 웹 사이트 복사기 또는 웹 사이트 리퍼 당신은 당신의 자신의 컴퓨터에 로컬 하드 드라이브에 인터넷에서 웹 사이트를 다운로드하실 수 있습니다. 웹 사이트 다운은 원래 웹 사이트 상대적인 링크 구조에 의해 다운로드 사이트를 배치. 다운로드 된 웹 사이트는 브라우저에서 HTML 페이지 중 하나를 개방함으로써 탐색 될 수있다.

HTML

그만큼 웹 그래버 로컬 하드 드라이브에 각 HTML 파일과 다운로드 및 복제를한다. 당신은을 사용할 수 있습니다 HTML 편집기 로컬 각 HTML 파일을 변경하거나 사용하기 온라인 HTML 편집기. 경우에 당신은 당신이 사용할 수 코더 없습니다 WYSIWYG - HTML 편집기 대신.

링크 변환

그들은 로컬로 작동 할 수 있도록 우리는 HTML 파일에있는 모든 링크를 변환, 오프라인, 대신 온라인 웹 사이트를 가리키는.

CSS 및 자바 스크립트

CSS 파일 및 자바 스크립트는 간단한 폴더 구조에 따라 다운로드하여 HTML 파일을 올바르게 참조 될.

이미지

다른 자산과 마찬가지로, 또한 이미지를 다운로드하고 로컬 참조.

인기있는 웹 사이트 다운 로더의 사용 사례

하드 드라이브에 웹 사이트를 복제 한 후에는 코드 편집기를 사용하여 웹 사이트의 소스 코드를 열거 나 단순히 오프라인 사용자가 선택한 브라우저를 사용하여 검색 할 수 있습니다. 사이트 다운 다수의 다른 목적으로 사용될 수있다. 그것은 사용하기 정말 간단합니다 웹 사이트의 다운로드 소프트웨어 아무것도를 다운로드하지 않고.

백업

당신은 웹 사이트가있는 경우, 당신은 항상 경우 서버 나누기에 웹 사이트의 최신 백업을해야하거나 해킹 얻을. 웹 사이트 다운 로더는 웹 사이트의 백업을 취할 수있는 가장 빠르고 쉬운 방법입니다, 그것은 할 수 있습니다 전체 웹 사이트를 다운로드.

오프라인 웹 사이트 다운

다운로드 웹 사이트 오프라인 당신의 미래 참조, 당신은 인터넷 연결 없이도 액세스 할 수있는, 말하다. 당신은 비행 또는 섬 휴가 때!

사이트 마이그레이션

당신은 당신의 웹 사이트의 소스 파일에 대한 액세스 권한이없는 호스팅 제공 업체에 또는 다른 이유로 액세스 할 수없는 경우, 단순히 사용 웹 사이트 리퍼 복사기 파일을 다운로드하고 새 서버로 귀하의 웹 사이트를 마이그레이션. 와 올바른 파일 권한을 설정하는 것을 잊지 마십시오 chmod를 계산기 마이그레이션 할 때.

배우기

당신은 웹 디자이너 또는 개발자 인 경우, 당신의 기술에 의해 리버스 엔지니어링 웹 사이트의 소스 코드에 의해 다음 단계로 설정 걸릴 전체 웹 사이트를 다운로드 새로운 UX 패턴 및 코딩 모범 사례를 배울 수. 전체 웹 사이트를 다운로드 학습을 시작합니다.

스크린 스크래핑

온라인 웹 사이트 다운 당신은 유용한 데이터를 추출 할 수 있습니다, 웹 페이지의 컨텐츠 정보 및 지식. 로컬 스크래핑 알고리즘을 실행하여 그들은 빠르고 부드럽게 실행!
