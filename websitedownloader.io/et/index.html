Veebileht Downloader Omadused

Veebileht Downloader, Veebileht koopiamasina või Veebileht Ripper lubab sul alla laadida veebilehtedel Internet kohalikul kõvakettal oma arvutisse. Veebileht Downloader korraldab allalaetud saidi originaal veebilehed suhteline link-struktuur. Allalaaditud kodulehel saab sirvida avades üks HTML lehekülgede brauseri.

HTML

The web kahmitseja võtab iga HTML faili ja allalaadimine ja kloonid seda kohalikul kõvakettal. Võite kasutada HTML editor teha muudatusi iga html faili kohapeal või kasutada HTML editor Internetis. Kui te ei ole kooder saab kasutada WYSIWYG HTML-redaktor selle asemel.

link Conversion

Me teisendada kõik lingid HTML faile nii, et nad töötavad kohapeal, off-line, asemel osutab veebisaidile.

CSS ja JavaScript

CSS ja JavaScript failid laaditakse alla lihtsa kausta struktuuri ja viidatud õigesti HTML faile.

pilte

Sarnaselt teiste varade, Samuti pildid alla laadida ja viidatud kohapeal.
