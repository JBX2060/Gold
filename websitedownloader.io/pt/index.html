Características site Downloader

Downloader site, Copiadora website ou website Ripper permite que você baixe sites da Internet para seu disco rígido local em seu próprio computador. Downloader site organiza o site baixado pelos sites originais link-estrutura relativa. O site baixado pode ser navegado através da abertura de uma das páginas HTML em um navegador.

HTML

o grabber web leva cada arquivo HTML e os downloads e clones lo em seu disco rígido local. Você pode usar um editor HTML para fazer alterações em cada arquivo html local ou utilizar um editor HTML on-line. No caso de você não é um programador pode utilizar um WYSIWYG HTML Editor de ao invés.

link Conversion

CSS e JavaScript

arquivos CSS e JavaScript será baixado sob uma estrutura de pastas simples e referenciados corretamente nos arquivos HTML.

imagens
