Caracteristici Site-ul Downloader

Site-ul Downloader, Site-ul Copier sau Site-ul Ripper vă permite să descărcați site-uri de pe Internet pe hard disk-ul local de pe propriul calculator. Site-ul Downloader aranjează site-ul descarcat de pe site-urile originale în raport cu structura link-ul-. Site-ul web descărcat poate fi accesat prin deschiderea uneia dintre pagini HTML într-un browser.

HTML

The Grabber web ia fiecare fișier HTML și download-uri și clone pentru a-l pe hard disk-ul local. Aveți posibilitatea să utilizați un editor HTML pentru a face modificări la fiecare fișier HTML local sau de a folosi un HTML editor on-line. În cazul în care nu sunteți un coder puteți utiliza un WYSIWYG HTML Editor in schimb.

legătură de conversie

Noi converti toate link-urile din fișierele HTML, astfel încât acestea funcționează la nivel local, deconectat, în loc de a fi îndreptat spre un site on-line.

CSS și JavaScript fișiere vor fi descărcate sub o structură de foldere simplu și se face referire în mod corect în fișierele HTML.
