Beneficii cheie

Cel mai simplu mod de a descărca de la archive.org, care este, de asemenea, cunoscut sub numele de mașină Wayback

platforma independentă

Interfața bazată pe web vă permite să utilizați downloader mașină Wayback direct în browser-ul pe orice sistem de operare și fără a descărca sau configura orice software.

Previews rapide

Wayback mașină downloader oferă o previzualizare rapidă a rezultatului de descărcare găzduit pe serverele noastre, fără a consuma spațiu pe disc prețios al computerului.

Simplitate

Wayback mașină downloader este foarte simplu și ușor de utilizat, dar vine cu o funcționalitate avansată, cum ar fi descărcarea de doar un subdirector sau anumite pagini dintr-un site (ca pagina de web downloader).

Wayback Caracteristici mașină Downloader

Wayback mașină Downloader vă permite să descărcați site-uri de la archive.org pe hard disk-ul local de pe propriul computer. Wayback Downloader aranjează site descărcate de către site-urile originale relative link-structura. Site-ul web descărcat poate fi accesat prin deschiderea uneia dintre pagini HTML într-un browser.

HTML

Wayback Downloader ia fiecare fișier HTML și download-uri și clone pentru a-l pe hard disk-ul local.

legătură de conversie

Noi converti toate link-urile din fișierele HTML, astfel încât acestea funcționează la nivel local, deconectat, în loc de a fi îndreptat spre un site on-line.

CSS & JavaScript

CSS și JavaScript fișiere vor fi descărcate sub o structură de foldere simplu și se face referire în mod corect în fișierele HTML.

imagini

La fel ca alte active, De asemenea, imaginile vor mai descărcat și se face referire la nivel local.
