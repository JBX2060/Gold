ویژگی های دانلود وب سایت

دانلود وب سایت, کپی وب سایت یا وب سایت چاک دهنده اجازه می دهد تا به شما دانلود وب سایت از اینترنت به دیسک سخت محلی خود را بر روی کامپیوتر خود. دانلود وب سایت مرتب سایت دانلود شده توسط وب سایت اصلی نسبی لینک ساختار. وب سایت دریافت می توان با باز یکی از صفحات HTML را در یک مرورگر مرور.

HTML

The دکمه های وب طول می کشد هر فایل HTML و دانلود و کلون آن را به درایو محلی. شما می توانید یک استفاده ویرایشگر HTML برای ایجاد تغییرات به هر یک از فایل اچ تی ام ال به صورت محلی و یا استفاده از ویرایشگر HTML آنلاین. در صورتی که شما یک رمز گذار شما می توانید یک استفاده نمی WYSIWYG-HTML ویرایشگر بجای.

تبدیل لینک

ما تبدیل تمام لینک ها در فایل های HTML به طوری که آنها به صورت محلی کار, آفلاین, به جای اشاره به یک وب سایت آنلاین.

CSS و جاوا اسکریپت

CSS و جاوا اسکریپت فایل های خواهد شد تحت یک ساختار پوشه ساده دانلود و به درستی در فایل های HTML اشاره.

تصاویر

مانند دیگر دارایی, همچنین تصاویر دانلود خواهد و به صورت محلی اشاره.

پس از شبیه سازی یک وب سایت به هارد دیسک شما می توانید از کد وب سایت های منبع را با یک ویرایشگر کد باز کردن و یا به سادگی آن را فهرست آنلاین با استفاده از یک مرورگر از انتخاب شما. دانلود سایت می توان برای اهداف مختلف مورد استفاده از چند. این واقعا ساده برای استفاده وب سایت دانلود نرم افزار بدون دانلود هر چیزی.
