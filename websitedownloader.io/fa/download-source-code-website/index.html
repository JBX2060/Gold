چگونه برای دانلود کد منبع از هر وب سایت?

Updated On: September 20, 2016
Written By: Debjit Saha

به منظور دانلود تمام یک وب سایت شما نیاز به آدرس وب سایت است (URL, برای مثال. apple.com یا google.com), اتصال به اینترنت و WebsiteDownloader.io

WebsiteDownloader.io یک ابزار عالی که شما اجازه می دهد تا دانلود کد منبع را از هر وب سایت که شامل فایل های HTML, دارایی های ثابت مانند JS (جاوا اسکریپت), CSS, عکس ها و اسناد PDF.

همه شما باید انجام دهید این است که آدرس وب سایت شما می خواهید برای دانلود وارد WebsiteDownloader.io و بعد از چند دقیقه, بسته به اندازه از وب سایت, شما یک پستی که حاوی کد منبع HTML از وب سایت دریافت کنید.

شما همچنین می توانید فهرستی از فایلهایی که فایل های فشرده شامل به طوری که شما به طور کامل از محتویات پستی مطمئن پیش نمایش.

جهان در حال حاضر دارای بیش از یک میلیارد وب سایت و کل جمعیت جهان است 7.4 میلیارد دلار و نوشتن این مقاله. این یعنی, وجود دارد 1 وب سایت برای هر 7 مردم بر روی زمین. با اينكه, این کار دقیقا همان که کار نمی کند اما آن را نمی خواهد اشتباه به برآورد مثل این – 1 وب سایت برای هر 7 مردم!
