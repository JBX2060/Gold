Características del Sitio Web Downloader

sitio web de descarga, Website Copier o Website Ripper le permite descargar sitios web de Internet en el disco duro local en su propio ordenador. Sitio web de descarga arregla el sitio descargado por los sitios web originales enlace de estructura relativa. La página web descargada puede ser consultada mediante la apertura de una de las páginas HTML en un navegador.

HTML

los web capturador utiliza todos los archivos HTML y las descargas y los clones en su disco duro local. Se puede utilizar una editor HTML para realizar cambios en cada archivo HTML de forma local o utilizar una editor HTML en línea. En caso de que usted no es un codificador puede utilizar una WYSIWYG HTML Editor en lugar.

Conversión de enlace

Convertimos todos los enlaces en los archivos HTML para que trabajen a nivel local, desconectado, en vez de apuntar a un sitio web en línea.

CSS y JavaScript

archivos CSS y JavaScript se pueden descargar de una estructura de carpetas sencilla y se hace referencia correctamente en los archivos HTML.

imágenes
